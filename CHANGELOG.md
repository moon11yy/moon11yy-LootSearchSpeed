## [0.1.0] - 2026-07-04

### Added

- Initial project structure.
- GitHub repository.
- Plugin information system.
- Centralized patch manager.
- Logging utility.
- Configuration system.
- BepInEx ConfigurationManager support.
- Vanilla container search research documentation.
- Clean patch architecture.
- Configurable container search delay multiplier.
- Configurable item reveal delay multiplier.
- Clean Harmony transpiler implementation.
- Tested on SPT 4.0.13.

### Changed

- Refactored plugin architecture.
- Improved project folder layout.
- Simplified patch registration.

### Notes

This is the first public release.
The mod modifies only vanilla search delays without skipping animations or changing game logic.
